A new paradigm in software development

What is it?

App Factory is the layman's tool for creating, designing, and delivering software applications. It breaks down the complexity of software engineering into simple, configurable building blocks. The user starts by creating a Blueprint, which can contain Documents to structure data, Routines to manipulate data, and Views to display data in convenient formats (like tables, lists, maps, and schedules). Then the user can create an Application based on the Blueprint. The Application is hosted by App Factory, which takes care of the complexity of maintaining servers, databases, etc. The user can then invite other users to their Application, with configurable roles and security. Over time, the user can modify the underlying Blueprint to adapt the Application to evolving needs.

When will it be available?

Currently, we are working on a prototype to demonstrate the concepts and capabilities that the real App Factory will offer. Formal development will being later this year. For updates, make sure to sign up for the newsletter. And in the meantime, feel free to explore the prototype.

How much will it cost?

App Factory is a free, open-source platform. Tech-savvy users will be able to setup and host an App Factory server using the source code. For the non-technical, there will be a hosting service similar to website hosting. Hosting small applications for individuals will likely be free, with reasonable storage and bandwidth limits. For businesses and enterprise, a comprehensive pricing model is forthcoming.
